Installation
============

Pypi
----

.. code:: shell

   pip3 install netblow --user

Upgrading
^^^^^^^^^

.. code:: shell

   pip3 install netblow --user -U

.. note::

   Legacy python (2.7) is not supported. 2020 is just around the corner, eh?


Docker
------

.. code:: shell

   docker pull quay.io/viniciusarcanjo/netblow
